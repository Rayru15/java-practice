## 순회(forEach)
`for` 문을 통한 순회와 `forEach`, `enhanced for` 를 통한 순회 모두 가능합니다.
```java
List<Integer> list = List.of(1, 2, 4, 9, 6);

// stream forEach
list.forEach(System.out::print);

// for loop
for (int i = 0; i < list.size(); i++) {
    System.out.print(list.get(i));
}

// enhanced for loop
for (int elem : list) {
    System.out.print(elem);
}
```
